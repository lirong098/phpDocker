# Docker部署php开发环境
## 结合真实的thinkPHP 5 为例
~~~bash
# 下载代码
git clone https://github.com/lirong098/phpDocker.git
# 进入项目
cd phpDocker
# 运行docker-compose
docker-compose up -d
# 浏览器输入网址测试
http://localhost/index.php/index/Index/moderndate
~~~
